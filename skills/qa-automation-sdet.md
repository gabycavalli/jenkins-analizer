# Skill Set: Senior QA Automation Engineer / SDET

## Test Automation Frameworks
* **Cypress Expert:** Diseño de arquitecturas escalables, manejo de comandos personalizados, interceptación de red avanzada y optimización de CI.
* **Playwright Specialist:** Ejecución paralela multi-browser, tracing, y gestión de contextos para pruebas de alta performance.
* **Legacy & Migration:** Experiencia en la transición de suites desde Selenium y TestCafe hacia frameworks modernos.
* **Mobile Testing:** Automatización de aplicaciones móviles y entornos responsivos.

## Technical Skills & Infrastructure
* **Languages:** TypeScript y JavaScript avanzado (Node.js).
* **Code Quality:** Implementación de **SonarQube** para el análisis estático de código en arquitecturas de microservicios.
* **API Testing:** Automatización de pruebas de integración con Postman, Supertest y validación de esquemas.
* **Version Control:** Estrategias complejas de Git (Branching, Merging, Rebase).

## Azure & DevOps
* **Azure Pipelines:** Creación de archivos YAML para la integración continua (CI) de pruebas automatizadas.
* **Azure DevOps:** Gestión del ciclo de vida de desarrollo (SDLC), trazabilidad de requerimientos y gestión de defectos.
* **Cloud Observability:** Monitoreo y telemetría utilizando Azure Application Insights y Log Analytics (KQL).

## Strategic Quality Leadership
* **SDET Mindset:** Enfoque en la calidad desde el diseño del código (Shift-Left Testing).
* **Performance Monitoring:** Configuración de dashboards de observabilidad en herramientas como New Relic para el seguimiento de errores en tiempo real.
